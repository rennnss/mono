## ✨ Key Features

*  **Aesthetic-Driven UI**: Gorgeous, track-adaptive background gradients that turn your currently playing song into a rich visual experience.
*  **Retro Vibes**: Unique, brutalist-inspired player states featuring animated spinning CDs and classic cassette tapes for that nostalgic feel.
*  **Local Library Management**: Easily browse your downloaded tracks with a built-in A-Z scrubber, search functionality, and a dynamic album grid view.
*  **Mixtapes**: Curate your own custom playlists and collections under the dedicated Mixtapes tab.
*  **Synchronized Lyrics**: Sing along with a clean, beautifully formatted, and easy-to-read lyrics view.
*  **Deep iOS Integration**: Fully supports native lock screen controls, Dynamic Island, and Control Center widgets.

## 🚀 Getting Started
